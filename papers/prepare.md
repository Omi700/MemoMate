# 智能冰箱贴提醒器（恋人专属版）初步方案

## 一、 项目定位

一款集**异地传情、教学助理、生活管家**于一体的智能自制冰箱贴。通过 3 英寸触摸屏与语音播报，为身在异地（或不同住）的教师女朋友提供每日工作提醒与情感陪伴。

## 二、 核心功能设计

### 1. 异地“爱心便签”与远程语音（恋爱互动）

- **功能：** 你可以通过微信小程序远程发送文本或录制原声语音。
- **交互：** 冰箱屏幕弹出信封动画，伴随提示音。她点击屏幕“签收”后，内容显示/播放，同时你的手机会收到“已读”通知。

### 2. “教案/班务”专属提醒（教学助理）

- **工作流提醒：** 针对小学语文老师繁杂的工作，在工作日早晨（如 7:10）自动播报：“今天是周三，记得带第二单元练习册，第一节是语文课。”
- **班务倒计时：** 屏幕常显重要节点，如“距离公开课还有 3 天”、“距离期末考试还有 10 天”。

### 3. 每日“诗词/金句”轮播（职业契合）

- **功能：** 屏幕底部或背景每日自动更新一首经典诗词或名家金句（可同步小学语文教材背诵篇目），提供教学灵感或晨读心情。

### 4. 冰箱生活管家

- **功能：** 临期食品提醒。在小程序登记“牛奶”，设置 3 天保质期，过期前当天早上语音提醒：“冰箱里的鲜牛奶快到期啦，记得喝掉。”

## 四、 系统架构与技术路线

### 1. 软件架构（零服务器成本方案）

- **前端（用户端）：** 微信小程序（使用**微信云开发**，利用其免费额度的云数据库、云存储，无需自建服务器）。
- **物联网云介质：** **巴法云 (Bemfa)** 或 **点灯科技 (Blinker)**（个人用户完全免费），利用 MQTT 协议实现小程序与 ESP32 的双向低延迟通信。
- **在线语音（TTS）：** 调用科大讯飞或百度语音的**免费个人额度 API**，由 ESP32 获取文本后实时转为音频流播放。

### 2. 硬件数据流向

$$\text{微信小程序（你输入）} \xrightarrow{\text{HTTPS/云数据库}} \text{物联网云平台 (MQTT)} \xrightarrow{\text{Wi-Fi}} \text{ESP32 (设备端)} \xrightarrow{\text{I2S/SPI}} \text{喇叭播放/屏幕显示}$$

## 五、 “研0”技术硬核攻关点（对个人能力的提升）

作为准研究生，在写这个项目时，可以重点啃下以下三个硬核技术，非常锻炼软硬件协同能力：

1. **高级 UI 框架库（LVGL）的运用：**

   不使用原始的画线/画方块代码，直接移植 **LVGL 嵌入式图形库**。在 3 英寸屏幕上实现丝滑的圆角按钮、滑动切页动画和精致的字体显示，大幅提升 UI 高级感。

2. **固件远程在线升级（HTTP OTA）：**

   异地恋必备。设备放在女朋友家，你人在学校，通过编写 OTA 代码，你可以远程将编译好的新固件上传到云端，让设备自动下载重启更新，无需现场插线调试。

3. **触摸校准算法与网络防断联机制：**

   - 编写**电阻屏四点校准算法**，将触摸物理坐标精确映射到屏幕像素坐标。
   - 加入 `WiFi.onEvent` 异步事件监听，确保家里 Wi-Fi 偶尔断网或路由器重启后，设备能自动重连，且断网期间本地 RTC 时钟仍能准时触发闹钟。

## 六、 后续实施步骤
